# README

This is a basic application that can add users to a tea subscription, remove them, and list subscriptions.

## Endpoints

### Subscribing
```/api/v1/customer_subs ```
### Unsubscribing
```/api/v1/customer_subs/:id ```
### All Subscriptions
```/api/v1/customers/:id/subscriptions ```
